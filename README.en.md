# CCloud Addon for Stremio

CCloud addon for Stremio app that allows you to watch movies and TV series.

## Features

- Browse movies and TV series
- Search functionality
- Filter by genre
- Full details including director, cast, and synopsis
- Multiple quality options (4K, 1080p, 720p, ...)

## Installation

### Automatic Installation (Recommended)

1. Open this link in your browser:
   ```
   https://ccloud-addon.onrender.com/
   ```
2. Click the **Install** button
3. Stremio will open and the addon will be installed

### Manual Installation

If automatic installation doesn't work:

1. Copy this link:
   ```
   stremio://ccloud-addon.onrender.com/manifest.json
   ```

2. Open the Stremio app

3. Go to the **Addons** section

4. Click **Add addon**

5. Paste the link and click the **Add** button

## Usage

After installation, two new sections will appear on Stremio's main screen:

- **CCloud Movies** - Movies catalog
- **CCloud Series** - TV series catalog

You can use the genre filter to find specific movies or series.

## Development

Built with Node.js and [Stremio Addon SDK](https://github.com/Stremio/stremio-addon-sdk)
